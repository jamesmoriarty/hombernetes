# Hombernetes

Repo for my k8s-gitops home lab.

## Diagram

![Arch](docs/arch.png)

## Features

* Microk8s (Host Storage)
* MetalLB
* ExternalDNS (AWS/Route53)
* Transmission (NAT/UDP)
* MiniDLNA (UPnP/UDP)
* Radarr & Jackett

## Requirements

```
Ubuntu 20.04.1 LTS
```
